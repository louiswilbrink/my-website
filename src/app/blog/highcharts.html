Getting Started with Highcharts and Angular

Date: June 1, 2015

Highcharts is a powerful graphing library that is particularly good for financial tech apps. The API reference is very well organized and detailed and the number of charts they offer make it a competitive offering versus D3. While it's not free, it's well worth the investment.

There is also an Angular module for Highcharts, called highcharts-ng, which wraps Highcharts into directives. Using this module allows you to source all of your data within the Angular ecosystem, and hand it off to a highchart directive. It's very helpful and straight-forward, however I didn't find a lot of examples on how to create many of the charts on the Highcharts Demo page. I've summarized a few of them in case other developers are experiencing this too.
