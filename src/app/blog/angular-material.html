Angular Material Tutorial

Date: March 2, 2015

Last year, Google announced Material Design and began to implement this design philosophy in all of its apps. If you use Google Inbox or Maps, you've already seen what a Material Design app looks and feels like.

Angular Material brings these UI components into the Angular world. I've written two apps using only Angular-Material components and have be pleased with the results: the apps are simple and intuitive. Plus, you get to leverage the fact that your users (if they use Google Apps) will already be familiar with the interactive components on the page

So let's jump right in

First, install Angular 1.3.x (required) and Angular-Material using your favorite method (bower, cdn, etc). Or you can have all of that taken care of for you by installing the following generator:

npm install generator-gulp-angular
yo gulp-angular myApp

Make sure to upgrade to the latest release on GitHub, not simply the offical pre-release version (0.6.0). The docs reflect the newest commits to the repo, as do all of the examples in this blog post.
